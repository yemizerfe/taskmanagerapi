# Task Manager Application

A simple and efficient task management web application that helps users organize, track, and manage daily tasks.

 Features
- Create, update, and delete tasks
- Track task completion
- Clean and minimal user interface
- REST API-based backend

 Tech Stack
- Frontend: React, CSS
- Backend: Node.js, Express.js
- Database: MongoDB

 Purpose
This project was built to improve personal productivity by providing an easy-to-use daily task tracking system.

Installation
1. Clone the repository  
   `git clone https://github.com/yemizerfe/taskmanagerapi.git`
2. Install dependencies  
   `npm install`
3. Run the application  
   `npm start`
